4.5.3 Create new Accumulator

importance: 2

Tasks:

Is it possible to create functions A and B so that new A() == new B()? 

function A() { ... }
function B() { ... }

let a = new A();
let b = new B();

alert( a == b ); // true

Answer:

Inside scripts.js : 

'use strict';

const obj = {};

function A() { return obj; }
function B() { return obj; }

alert( new A() == new B() ); // true
