8.1.3 Where does it write?

importance: 5

Tasks:

We have rabbit inheriting from animal.

If we call rabbit.eat(), which object receives the full property: animal or rabbit? 

let animal = {
  eat() {
    this.full = true;
  }
};

let rabbit = {
  __proto__: animal
};

rabbit.eat();

Answer:

Inside scripts.js : 

'use strict';

const animal = {
  eat() {
    this.full = true;
  }
};

const rabbit = {
  __proto__: animal
};

rabbit.eat(); // rabbit.full = true;
