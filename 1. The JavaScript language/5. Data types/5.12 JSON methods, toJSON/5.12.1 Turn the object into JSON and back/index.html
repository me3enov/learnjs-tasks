5.12.1 Turn the object into JSON and back

importance: 5

Tasks:

Turn the user into JSON and then read it back into another variable. 

let user = {
  name: "John Smith",
  age: 35
};

Answer:

Inside scripts.js : 

'use strict';

const user = {
  name: "John Smith",
  age: 35
};

const user2 = JSON.parse(JSON.stringify(user));
