2.5.1 String quotes

importance: 5

Tasks:

What is the output of the script? 

let name = "Ilya";

alert( `hello ${1}` ); // ?

alert( `hello ${"name"}` ); // ?

alert( `hello ${name}` ); // ?

Answer:

Inside scripts.js : 

'use strict';

const name = 'Ilya';

alert( `hello ${1}` ); // => alert message "hello 1"
alert( `hello ${'name'}` ); // => alert message "hello name"
alert( `hello ${name}` ); // => alert message "hello Ilya"
