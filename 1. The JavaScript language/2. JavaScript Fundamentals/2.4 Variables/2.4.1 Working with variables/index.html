2.4.1 Working with variables

importance: 2

Tasks:

1. Declare two variables: admin and name.

2. Assign the value "John" to name.

3. Copy the value from name to admin.

4. Show the value of admin using alert (must output “John”).

Answer:

Inside scripts.js : 

'use strict';

let admin;
let name;

name = 'John'
admin = name

alert( admin )
