2.13.2 Which values does the while loop show?

importance: 4

Tasks:

For every loop iteration, write down which value it outputs and then compare it with the solution.

Both loops alert the same values, or not?

1.
The prefix form ++i: 

let i = 0;
while (++i < 5) alert( i );

2.
The postfix form i++: 

let i = 0;
while (i++ < 5) alert( i );

Answer:

Inside scripts.js : 

'use strict';

let i = 0;
while (++i < 5) alert( i ); // => 1; => 2;  => 3; => 4;

let j = 0;
while (j++ < 5) alert( j ); // => 1; => 2;  => 3; => 4; => 5;
