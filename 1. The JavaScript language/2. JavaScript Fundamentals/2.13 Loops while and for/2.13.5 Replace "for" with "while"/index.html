2.13.5 Replace "for" with "while"

importance: 5

Tasks:

Rewrite the code changing the for loop to while without altering its behavior (the output should stay same). 

for (let i = 0; i < 3; i++) {
  alert( `number ${i}!` );
}

Answer:

Inside scripts.js : 

'use strict';

let i = 0;
while (i < 3) {
  alert( `number ${i}!` );
  i++;
}
