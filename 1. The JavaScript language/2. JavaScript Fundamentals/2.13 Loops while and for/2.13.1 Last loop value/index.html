2.13.1 Last loop value

importance: 3

Tasks:

What is the last value alerted by this code? Why? 

let i = 3;

while (i) {
  alert( i-- );
}

Answer:

Inside scripts.js : 

'use strict';

let i = 3;

while (i) { // i = 3; i = 2; i = 1; i = 0 => i = false
  alert( i-- ); // => 3'; => 2;  => 1;
}
