2.8.1 The postfix and prefix forms

importance: 5

Tasks:

What are the final values of all variables a, b, c and d after the code below? 

let a = 1, b = 1;

let c = ++a; // ?
let d = b++; // ?

Answer:

Inside scripts.js : 

'use strict';

let a = 1, b = 1;

let c = ++a; // c = 2
let d = b++; // d = 1
