5.3.1 Uppercase the first character

importance: 5

Tasks:

Write a function ucFirst(str) that returns the string str with the uppercased first character, for instance: 

ucFirst("john") == "John";

Answer:

Inside scripts.js : 

'use strict';

const ucFirst = (str) => {
  if (!str) return str;
  return str[0].toUpperCase() + str.slice(1);
}

alert( ucFirst("john") ); // John
